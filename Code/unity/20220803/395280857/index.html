【Unity展厅】 AI识别语音机器人

发表于 2022-08-03 分类于 Code ， unity展厅 Valine：
 简介：用Unity做展厅类的软件,有滑屏、翻书、单切页、播放视频、网页浏览、照片墙、手势识别、互动全息投影、中控、多指触摸屏幕、签名、N连屏、飞屏、AI语音、AR/VR，等等。  注意:资源需要动态替换！Unity版本：2018.2.20f
 实现：语音识别，智能回复。系统:Win10 插件： RTVoice，支持语音转文字,自带主持人,unity 转.net 4.0

1： RTVoice插件导入

1 导入unity,右键添加RTVoice - RTVoice  和 SpeechText
2 修改SpeechText中的culture为中文 ch|cn ，mode = speak native 
3 Text 写个 你好   Test-Drive点击speak 就可以听到声音了
4 RTVoice语音管理， 自己添加系统语音库
6 没声音自行百度打开系统语音

插件统一下载地址 ：

2： Unity3种识别语音

👉 短语识别

1 KeywordRecognizer  ：监听语音输入并尝试确定说出的短语。关键字识别
2 GrammarRecognizer  ：KeywordRecognizer的补充判断更复杂的语法

👉 听写识别

1 DictationRecognizer ：监听语音输入并尝试确定说出的短语。
2 不能同时使用短语识别和听写识别
3 其实就是将用户的语音转换为文本,再和我们的文本比对结果差不多就读出来；
4 自行启动系统听写功能哦；
