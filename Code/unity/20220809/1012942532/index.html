【Unity展厅】 小程序AR,webAR方案

发表于 2022-08-09 分类于 Code ， unity展厅 Valine：
 简介：用Unity做展厅类的软件,有滑屏、翻书、单切页、播放视频、网页浏览、照片墙、手势识别、互动全息投影、中控、多指触摸屏幕、签名、N连屏、飞屏、AI语音、AR/VR，等等。  注意:资源需要动态替换！Unity版本：2018.2.20f;插件:opencv for unity (官网有免费的)
 需求： 扫描识别图 - 生成模型 - 播放动画声音 - 拍照纪念

1： 方案介绍

 1 对接SDK我这边选了2家 ： easyAR,Kivicube
 2 自己接web支持AR.js - aframe - three.js
 3 别人封装的mindAR，基础还是AR.js这个库

》 AR.js库 ：
》 Mind-AR库 ：
》 Mind-AR演示地址 ：

2： 试玩MindAR

  1 要使用webAR，要在自己的电脑上搭建一个网站
  2 还要是https的前缀
  > windows10为例：右击此电脑--属性--控制面板--网络和Internet--程序--启用或关闭Windows功能
  --勾选Internet Information Services 所有选项--确定，等待操作完成
  > 右击此电脑--管理--服务和应用程序--网站右键添加。

  3 模型用.glb格式，识别图片https://hiukim.github.io/mind-ar-js-doc/tools/compile
  4 生成一个targets.mind的特征点识别图；可以多图

 1 由于项目周期太短，自己接的问题比较多
 2 所以选择对接别人平台的。
 3 我用的是Kivicube这个平台。
