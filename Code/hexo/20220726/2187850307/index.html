【hexo建站】 一: git + hexo + next 下载与安装

 简介：使用github+hexo搭建网站,我使用的主题是next | next版本为 7.8；hexo版本5.4.2；

1： Git安装

点击进入：Git下载地址1————Git下载地址2： 如果下载很慢：国内Git镜像站点 <—翻到最下面最新的下(Git-XXX-64-bit.exe) ；下完默认下一步就可以安装完成！ 不放心可以打开cmd命令输入

点击进入：git上传代码到hub

git 不是内部命令 ? 点我解决吧

2： Hexo安装

点击进入：安装Node：；要先下载NodeJS因为要使用npm命令下载Hexo

命令 描述
npm Node.js 包管理器，可以下载包、安装包、卸载包、更新包、上传包等。

老规矩一顿下一步就可以安装成功了

2.1： 选择(Hexo)建站路径

 我安装的是在D盘，然后在D盘某文件夹右键点 git bash here或者在终端打开

点击进入：HEXO官方教程；

点我打开,安装hexo顺序 复制<---前面的代码在站点文件夹中右键点 git bash here输入....

2.2： Hexo常用命令

命令 描述
hexo clean 清理生成的网页，路径blog–>pulibc
hexo g 编译,生成网页
hexo s 启动网页,本地地址http://localhost:4000/
hexo d 上传网页到git
Ctrl c 中断
hexo new XX 生成文章，路径：blog–source–_porsts
hexo new page XX 生成页面，路径：blog–source XX
hexo new draft XX 生成草稿

3： nexT安装

点击进入：Next官方教程；

 两种方式进行安装我使用第二种：
 1：git clone https://github.com/theme-next/hexo-theme-next themes/next
 2：下载https://github.com/theme-next/hexo-theme-next,将包解压到blog--themes里。
 <————————————————————————————————————————————————————————————————————————>
 3：修改主题hexo-theme-next；改文件夹名字next 。
 4：然后修改站点配置文件_config.yml。主题也有一个config 。
 5：找到blog--_config.yml,搜索：theme:  将 theme: XXXXXX 换成next 。
 6：此时编写命令hexo g , hexo s;
 7：网址刷新一下就可以看到next主题了。
