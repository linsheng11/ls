 3 搜 back2top 设置返回顶部的按钮

👉 文章太多,让主页更简洁

  1 主题配置文件中 搜 Post Settings 🐾 scroll_to_more: true #选取博客正文<!--more-->前的内容
  2 在位置想要显示的部分后添加<!--more--> 即可；
  3 可以在站点配置文件中 修改index_generator - per_page ： 9 个 首页只显示9个内容 自己调节哦

  2 打开blog\themes\next\layout\_macro | post.swig
  3 搜 far  把它改成fa 格式即可
