【hexo建站】 五: 域名申请;DNS解析;CDN加速网站

发表于 2022-07-30 分类于 Code ， hexo建站 Valine：
 简介：github名字太丑决定换一个，国内访问速度有点慢需要CDN加速

1： 安装谷歌浏览器插件

  1 Gooreplacer 为了拦截/重定向 URL 
  2 打开chorm在应用商店安装Gooreplacer
  3 然后再Gooreplacer主页设置匹配模式：www.google.com/recaptcha  
  4 目标地址：recaptcha.net/recaptcha

2： 注册域名

👉 注册

注册地址 ：
10分钟邮箱注册地址 ： 

  1 输入XXXX.tk 点击检查可用性
  2 打勾,点完成
  3 选择12个月，下一步
  4 注册一个10分钟邮箱，输入刚注册的邮箱
  5 邮箱中收到消息了点击
  6 此订单在安全环境中提供，以帮助防止欺诈您当前的 IP 地址 ( 165.154.253.10 ) 正在被记录。
  7 注意你的IP地址,正常填写，翻墙的注册IP所在地方的人物信息
  8 登入点击Services的下拉菜单，选择My Domains 就能看见我们刚才注册的域名了，点击Manage Domain

👉 DNS解析

  1 配置DNS, 点击 Manage Freenom DNS
  2 解析中要等待
Name Type TTL Target
A 3600 185.199.108.153
A 3600 185.199.109.153
A 3600 185.199.110.153
A 3600 185.199.111.153
www CNAME 3600 Name.github.io

👉 网站配置域名

  1 站点配置文件url 改为XXX.tk
  2 \blog\source  | 新建 CNAME 无后缀   里面添加域名地址
  2 提交hexo d 
  3 登入github在你的仓库中点settings - pages -Custom domain 
  4 就可以看到域名已经改变，Enforce HTTPS 打勾强制https

👉 CDN加速

  1 由于是tk域名,cdn加速在国内需要备案。
  2 所以选择jsdelivr加速。
  3 近期cdn.jsdelivr.net被墙了，搜了一下
  4 可以使用https://gcore.jsdelivr.net/gh/xx1/xx2/xx.img
命令 描述
gh github简写
xx1 用户名
xx2 仓库名
xx.img 路径 
  5 后面可能也会被墙且用且珍惜
