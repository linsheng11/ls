👉 加载前提介绍

 1 由于项目会有大量图片视频，都打进Resources
 2 考虑到复用性，图片可能经常需要替换
 3 所以把资源都放在StreamingAssets下，基本都是pc所以不考虑手机上路径问题了
  • 大多数平台（Unity Editor、Windows、Linux 播放器）使用Application.dataPath + “/StreamingAssets”,
  • macOS 播放器使用Application.dataPath + “/Resources/Data/StreamingAssets”,
  • iOS 使用Application.dataPath + “/Raw”,
  • Android 使用压缩包内的文件APK
  • /JAR 文件，”jar:file://“ + Application.dataPath + “!/assets”.
